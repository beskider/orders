# Orders

The fullstack application managing orders.

## Technologies

* [Node.js](https://nodejs.org/en/)
* [Nodemon](https://nodemon.io/)
* [React](https://pl.reactjs.org/)
* [Bootstrap](https://getbootstrap.com/)

## Running

```
	$ ./npm start
```
